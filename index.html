GAUS 0.0.52

2019-07-02

  • Administrators can now disable/enable users in User Administration page.
  • Column picker added in device list UI.
  • Full support for Alphanum version type.
  • Improved e-mail invitation message to new users.
  • Terms of use updated.
  • Documentation links to Swagger added in UI.
  • Added status "Registered" for devices.

GAUS 0.0.48

2019-06-10

  • Added User Admin page. New User Administrator role added. User with this role can add normal users.
  • New package list page in UI
  • firmware-version is not a mandatory parameter anymore.
  • Added device detail page.

GAUS 0.0.44

2019-04-16

  • In earlier versions only firmware could be updated. In this version support is added for multiple update types.
  • Support for Grafana is added. This feature does not come out of the box and must be requested to GAUS admin explicitly.

GAUS 0.0.42

2019-03-28

  • In this release users can edit an update as well as pause/un-pause an update.
  • New Login page with black background added for a cooler look and feel.

GAUS 0.0.39

2019-03-11

  • What-if support added for updates in the UI. This means that during an update creation the user will get instant feedback on how many devices the update will hit.
  • Reference implementation and developer documentation are publicly available and can be accessed on GitHub: https://github.com/gaus-fleet-lab
